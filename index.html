The Newman design system

Logo, color, typography, spacing and components for Newman · Soluciones en Energía. A light, purple-forward, flat system — restraint over decoration. This page is the system documenting itself.

01 · The Logo

Isotype + wordmark.

Five descending diagonal bars form an abstract “N” that reads as solar rays and momentum — the isotype, which always keeps its gradient. The wordmark is a bold neo-grotesque that flips to suit the surface. The isotype's navy end can camouflage on dark; on dark surfaces use the reversed all-white lockup, never the gradient.

Primary — navy wordmark + gradient isotype, on light surfaces.

Reversed — all white, on navy / photography. Use this in the nav and footer.

Isotype

Standalone for avatars, favicons, app icons. Min 16px.

Clear space & min size

Clear space = one isotype-bar height on all sides. Min lockup 120px wide.

03 · Typography

Montserrat + Inter.

Montserrat carries display, headings and labels; the signature is weight 300 at large sizes — architectural, never shouted. Inter handles body. Never Montserrat below 12px; never tint body text magenta.

06 · Do & Don't

Protect the system.

Do

  • Lead with canvas / surface / navy; let solid accent be the rare highlight.
  • Set display headlines in Montserrat 300 at large sizes.
  • Prefix every section label with the 4px magenta square.
  • Keep rounding generous — 100px pills, 64px media, 24px cards.
  • Use the reversed white logo on dark surfaces.

Don't

  • Don't add chrome gradients — gradient belongs only in the logo isotype.
  • Don't tint body text magenta — it fails contrast.
  • Don't use bold/semibold for display headlines.
  • Don't add drop shadows — stay flat and tonal.
  • Don't put the gradient isotype on navy — it camouflages.
