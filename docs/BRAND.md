# Newman — Brand Quick Reference

> Full system in [`brandbook.html`](./brandbook.html). This is the working cheat-sheet.

## Essence
- **Name:** Newman · *Soluciones en Energía* · newman.re
- **What:** finance + commercialize solar generation and behind-the-meter storage, Mexico & LATAM
- **Tagline:** Shape the Future · Join the New Era · *Let's light it up*
- **Voice:** visionary · solid · bold · human. Aspirational, backed by evidence. Bilingual EN/ES. Never hype, jargon, or corporate-cold.

## Logo
- **Isotype:** 5 descending diagonal bars — abstract "N" / solar rays / momentum. **Always keeps the gradient.**
- **Wordmark:** bold neo-grotesque caps, flips black↔white. ® on official material.
- **Clear space:** 1 bar height on all sides. **Min size:** lockup 120px / 32mm, isotype 16px.
- **Don't:** rotate, stretch, recolor, shadow, retype, or place on busy/low-contrast.

## Color
| Token | HEX | Role |
|-------|-----|------|
| Newman Navy | `#191B4D` | primary · gradient origin |
| Newman Magenta | `#B80E65` | primary · gradient apex · accent |
| Magenta-Ink | `#8A0A4B` | **contrast-safe magenta for text on light** |
| Indigo / Plum / Grape / Berry | `#211A4E` `#3A1852` `#621558` `#981060` | gradient stops |
| Ink Black | `#0A0A0C` | dark canvas |
| Paper | `#F5F5F4` | light surface |

**Signature gradient:** `linear-gradient(105deg, navy → magenta)` — navy→magenta only, never reversed.
Interpolate in **OKLCH** where supported. Flat fallback: `#621558`.
Ratio: ~80% navy/black/white, magenta + gradient as accent. **Never tint body text magenta.**
**Never run the gradient under body text.**

## Type
- **Display / headlines:** Space Grotesk — uppercase, wide tracking (0.04–0.5em), never < 12px.
- **Body:** Inter — sentence case, default tracking.

## Graphic kit
Rays (isotype bled off-edge) · the gradient **Brand Bar** (the fast brand signal) · stepped diagonal bar motif · wide tracked-caps lockups.

## Photography
Cinematic, dark, desaturated near-black. Optional cool navy shadow tint. Generous negative space for type. Real projects over stock.

## Proof points
+$90M USD capital placed · +200 EPC partners · 54,512 t CO₂ reduced / year.
